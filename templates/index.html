Intro to Ninkasi

Fun Fact: In the myth of Mesopotamia, Ninkasi is the ancient Sumerian tutelary goddess of beer.

Advanced Algorithms

Techniques for building Ninkasi involve Natural Language Processing (NLP), Latent Semantic Analysis (LSA), matrix factorization (SVD++) and Restricted Boltzmann Machine (RBM).

Multifunctional

Two independent systems allow users to get recommendations via different ways. You can either find beers similar to your favorite one, or see others with a similar taste also love to drink.

Reliable Data Source

The data comes from ratebeer.com, an influencial global beer forum. 280,000 reviews, 1269 different beers, and 17,000 users' ratings are used to build the core engines of Ninkasi.

Workflow

The entire project last for 3 weeks.

  • I. Web scraping

    280,000 beer reviews, info and ratings are scraped from ratebeer.com using Python Scrapy package

  • II. Data Cleaning

    Beer info and rating data are cleaned. Text mining, tokenization, spell check and lemmatization are performed on beer reviews

    Two algorithms are applied. A Matrix factorization method-SVD++ and an unsupervised deep learning technique-Restricted Boltzmann Machine (RBM). The results from two methods are combined to reconstruct user-item matrix.
