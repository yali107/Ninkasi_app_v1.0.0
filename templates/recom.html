RECOMMENDER I. SIMILAR TASTE

Find beers similar to your favorite.

Recommender I is built upon 280,000 user reviews. The techniques include Natural Language Processing (NLP), Term Frequency–Inverse Document Frequency (TF-IDF), and Latent Semantic Analysis (LSA). This is a content-based recommendation system.

Ready to get some beer recommendation? Enter "SIMILAR TASTE" tab, type in the name of your favoriate beer and click "Submit"!

RECOMMENDER II. SIMILAR PEOPLE

Find beers from other people like you!

Recommender II is built upon ratings from 17,000 users. Two algorithms are used to deal with the sparsity of the user-item matrix. One is Matrix Factorization containing implicit feedback information (SVD++). Another is Restricted Boltzmann Machine (RBM), an unsupervised deep learning technique. Two method are stacked to give the final user-item matrix.

In the "SIMILAR PEOPLE" tab, enter some beer names and rate them from 1 to 20, and click "Submit"!

Note: (1) You can input up to 10 beer-ratings, and the input order does not matter. (2) If you input same beer more than once, the latter will be taken into account. (3) It is ok to skip lines
